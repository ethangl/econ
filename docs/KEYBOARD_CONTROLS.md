# Keyboard Controls

## Map Modes (`MapView`)

- `1`: Political
- `2`: Biomes
- `3`: Market
- `0`: Channel Inspector

## Rendering Debug (`MapView`)

- `O`: Cycle Channel Inspector debug view (auto-enters Channel Inspector if needed)
- `P`: Toggle on-screen ID probe

## Simulation Time (`GameManager`)

- `Backspace`: Pause/unpause simulation
- `-`: Decrease simulation speed preset
- `=`: Increase simulation speed preset

## Camera (`MapCamera`)

- `W`, `A`, `S`, `D`: Keyboard pan
- Arrow keys: Keyboard pan
- Mouse wheel: Zoom
- Middle mouse drag: Pan
- Hold `Space` + left-drag: Pan (laptop-friendly)

## UI Shortcuts

- `E`: Toggle economy panel
- `Escape`: Close market inspector panel (when open)
- `Escape`: Clear selection panel selection
- `Escape`: Hide economy panel (if visible)
